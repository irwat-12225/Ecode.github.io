# 🏔️ ECODER - Plateforme Officielle du Secteur Ruwenzori

## 📋 À propos du projet

**ECODER** (Engagement pour la Construction et le Développement du secteur Ruwenzori) est une initiative citoyenne visant le développement durable et la prospérité du secteur Ruwenzori à Beni, République Démocratique du Congo.

Cette plateforme web a été créée pour :
- Présenter l'initiative ECODER
- Mobiliser les ressortissants et la population
- Promouvoir l'éducation universitaire
- Lutter contre le chômage
- Faciliter les opportunités de collaboration

## 🎯 Objectifs

- ✅ Mobilisation des ressortissants
- ✅ Promotion de l'éducation
- ✅ Création d'opportunités d'emploi
- ✅ Développement durable du secteur
- ✅ Collaboration avec les autorités locales

## 🚀 Comment utiliser

1. **Ouvrir le site**
   ```bash
   # Ouvrir index.html dans un navigateur web
   ```

2. **Navigation**
   - Utilisez la barre de navigation pour accéder à toutes les pages

3. **Adhérer à l'initiative**
